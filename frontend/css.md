# css

Cascading Style Sheet 层叠级联样式表



## 什么是css

1. css选择器（重点+难点）

2. 美化网页（颜色，字体，宽度，高度，背景图片，文字、阴影、超链接、列表、渐变。。）

3. 盒子模型：

   ![image-20210113082847390](./resources/image/box-model.png)

4. 浮动：广告弹窗

5. 定位：广告弹窗

6. 网页动画（特效效果）

参见，菜鸟教程，**css教程**和**css3教程**

css 3.0	圆角，阴影，动画。。。浏览器兼容性
